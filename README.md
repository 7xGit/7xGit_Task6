Task 6 - Simon Game Challenge 
=======================

Welcome to the **Simon Game Contest**! In this game, you replicate a sequence of button presses generated by the system. Sounds simple? Here's the twist—some code is **missing**, and it's up to **you** to fix it! 🛠️

🧠 Challenge Overview
---------------------

Your goal is to restore the game's functionality by completing missing code in index.js. Pay close attention to game logic!

### What's the Twist?

*   The index.js file has **missing logic** that needs fixing.
    
*   Decide how the game handles **incorrect user input**.
    
*   Some functions are **not called**. Can you wire them up?
    

🎲 Game Description
-------------------

The game generates a **random sequence of colors**. Your task is to click the buttons in the **correct order**. Each level, the sequence gets **longer**. Can you keep up? The scoring system needs your help to work smoothly.

✅ Your Task
-----------

*   Fix missing logic in index.js. What happens on errors?
    
*   Ensure the game **restarts** after a mistake. How will you do it?
    
*   Spot and implement any **missing key function**.
    

🚀 Clone and Run Locally
------------------------

Set up the project on your local machine using the terminal:

1.  Clone the repository: 
    ```bash
    git clone https://github.com/7xGit/7xGit_Task6.git
    ```
    
2. Navigate to the project folder:  
    ```bash
    cd repository
    ```
    
3. Run the game:

    ```bash
    open index.html  # macOS
    xdg-open index.html  # Linux
    start index.html  # Windows
    ```
    

🎯 How to Play
--------------

*   Press **any key** to start the game.
    
*   Click buttons in the order of the **color sequence**.
    

🏆 Winning Conditions
---------------------

Win by restoring the game logic to work **flawlessly**. Focus on:

*   **Game resets**
    
*   **Sequence generation**
    
*   **Pattern comparison**
    

May the **best coder** win! ✨

⚙️ Game Mechanics
-----------------

*   nextSequence(): Generates the next color in the sequence.
    
*   checkAnswer(): Compares user input with the pattern (missing key parts).
    
*   playSound(): Plays sound on button press.
    
*   animatePress(): Adds visual effect on button click.
